# 샘플 문서 안내

이 문서는 `content/writing` 폴더의 기본 작성 예시입니다.

## 포함 요소

- Frontmatter 기반 메타데이터(`title`, `description`, `date`, `tags`)
- 본문 섹션 구조(`#`, `##`)
- 리스트와 코드 블록

## 예시 코드

```ts
const message = 'Sample post is ready.'
console.log(message)
```

## 메모
